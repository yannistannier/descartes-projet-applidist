# Web Project (Spitch Application)

## Introduction

Spitch is a video quiz application that allows users
discover, create and share video content. The application is now available on
Android and IPhone.

## Technical Stack 
* Django (Python).
* Spring (Jee).
* Docker.
* AWS Elastic Beanstalk.
* AWS CloudFormation.
* AWS Lambda.
* AWS DynamoDB.
* AWS SQS.
* AWS Transcoder.
* AWS StepFunction.
* AWS S3.
* React Native.
